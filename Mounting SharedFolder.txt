DO NOT FORGET TO install Vbox guests additions from virtual cd (irtualbox devices)


- temporary mount:
Sudo mount -t vboxsf nameofHostFolder ~/path/toVirtualFolder  (FOLDERS MUST BE DIFFERENT NAMES)

- permanent mount:
open this file: /etc/fstab
add this: nameofHostFolder /home/user/new vboxsf defaults 0 0


Shared folders when mounted can be accessed in linux mostly at : /media/sf_sharedvbox

- grant access to non root user:
sudo usermod -aG vboxsf $(whoami)
-or prefer:
sudo adduser (your user name here) vboxsf

make sure vboxsf group is added to the media subfolder, if not:
sudo chgrp vboxsf /media/sf_sharedvbox
